# AI Evals

This directory stores periodic AI evaluation artifacts.

## What to store here

- Baseline snapshots before major changes
- Weekly KPI snapshots (pre/post comparisons)
- Short qualitative review notes for regressions

## Naming

Use UTC date prefixes:

- `YYYY-MM-DD-AI-ADVISOR-PHASE0-BASELINE.md`
- `YYYY-MM-DD-AI-ADVISOR-WEEKLY-KPI.md`

## Minimum weekly entry

1. KPI query output (or summarized values)
2. Denominators (`typed_turns`, `identifier_turns`, `chart_turns`, `write_turns`)
3. Pass/fail notes vs prior week
4. Action items (if any)
