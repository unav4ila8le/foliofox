# AI Docs

This folder contains Foliofox AI product and implementation docs.

## Contents

- `AI-ADVISOR-ROADMAP.md`: strategic roadmap for AI Advisor
- `AI-ADVISOR-PHASE0-EVALS.md`: operational runbook for weekly Phase 0 evals
- `evals/`: dated evaluation snapshots and weekly KPI notes

## Convention

- Keep roadmap/runbook docs in `docs/ai/`.
- Keep time-based evaluation artifacts in `docs/ai/evals/`.
- Use UTC dates in filenames for snapshots.
- Start weekly notes from `docs/ai/evals/TEMPLATE-AI-ADVISOR-WEEKLY-KPI.md`.
